# StoryCMS Kit

StoryCMS package for AdonisJS

## Installation

```shell
# NPM
npm i @story-cms/kit

# or Yarn
yarn add @story-cms/kit
```

### Develop

```
npm run dev
```

## Contributing

This project happily accepts contributions.

If you have a question or found a bug, feel free to
[open an issue](https://github.com/story-cms/kit/issues).
